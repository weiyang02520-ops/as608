# AS608 指纹打卡机

这是一个基于 STM32F103C8Tx 的指纹打卡项目，使用 AS608 指纹模块完成用户识别，通过 OLED 和 4x4 矩阵键盘完成本机交互，并使用 AT24C32 EEPROM 保存打卡记录、管理员密码和累计时长数据。项目由 STM32CubeIDE / STM32CubeMX 生成基础工程，业务代码主要集中在 `Core/Src` 和 `Core/Inc`。

## 主要功能

- 指纹录入、搜索、删除和清空
- 本机 OLED 菜单管理
- 管理员密码校验和修改
- 用户姓名写入 AS608 记事本区域
- 上班 / 下班打卡记录
- 按用户累计工作时长
- 蓝牙串口同步时间
- 蓝牙串口导出排行榜数据

## 硬件组成

| 模块 | 连接/用途 |
| --- | --- |
| STM32F103C8Tx | 主控 |
| AS608 | 指纹识别模块，`USART1`，57600 baud |
| 蓝牙串口模块 | 手机通信，`USART3`，9600 baud |
| AT24C32 | I2C EEPROM，地址 `0x57` |
| OLED | GPIO 模拟 SPI 显示 |
| 4x4 矩阵键盘 | PA0-PA7 |
| RTC | 使用 LSI 时钟和备份寄存器 |

## 目录结构

```text
Core/
  Inc/        业务头文件和 CubeMX 生成头文件
  Src/        主程序、指纹、按键、OLED、RTC、串口等代码
  Startup/    STM32 启动文件
Drivers/      STM32 HAL 和 CMSIS 驱动
as608.ioc     CubeMX 配置文件
*.ld          链接脚本
```

## 构建方式

推荐使用 STM32CubeIDE 打开本工程：

1. 导入项目目录。
2. 确认目标芯片为 `STM32F103C8Tx`。
3. 使用 Debug 或 Release 配置构建。
4. 通过 ST-Link 下载到开发板。

`Debug/` 和 `Release/` 是本地编译输出目录，已经在 `.gitignore` 中排除。

## 蓝牙指令




## 当前状态

当前代码可以作为课程设计、嵌入式实验或二次开发项目使用。工程里已经包含 HAL/CMSIS 驱动、CubeMX 配置和链接脚本，克隆后用 STM32CubeIDE 打开即可继续开发。

## 已知注意点

- EEPROM 记录区、管理员密码区和累计时长区建议进一步整理，避免长期使用后地址重叠。
- 当前打卡时长主要按当天秒数计算，跨天打卡场景还需要补充处理。
- 指纹 ID 当前按有效模板数量递增，删除中间 ID 后再次新增时需要注意覆盖风险。
- 串口接收解析还有优化空间，后续可改为按二进制包格式解析。
